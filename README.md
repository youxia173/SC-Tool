# SC 小工具 v1.4.0

一款用于《星际公民》（Star Citizen）的中文输入与截图工具，提供中文输入叠加窗口、IME 词表转译、剪贴板粘贴发送、可选中英双语发送、游戏内截图、聊天区 OCR 英译中、自定义快捷键和系统托盘等功能。

按下唤醒键后，程序会同时打开游戏聊天框和中文输入窗口。使用系统输入法完成中文输入后，再次按下 `Enter` 即可转译并发送内容。开启「双语发送」时，还会调用所选翻译接口生成英文，并与中文转译一并粘贴发送。

## 更新说明（v1.4.0）

- 输入繁体中文时，发送前自动本地转为简体再转译发出（无需联网 / 调用 API），方便台湾等地用户。
- 新增 DeepSeek 翻译引擎（Flash / Pro 可选，共用一把 API Key，国内直连）。

## 更新说明（v1.3.0）

相对 v1.2.0，本版本主要新增与调整：

### 聊天区 OCR 与英译中

- 支持框选游戏聊天区域，识别英文聊天内容；
- OCR 引擎可选：系统 OCR（免费）或百度云高精度 OCR（付费，需单独 API Key，与翻译密钥不同）；
- 按玩家名分行整理，并对每条正文做英 → 中翻译；
- 识别与翻译结果可显示在可配置的提示浮层中，并复制到剪贴板；
- 新增「识别聊天区」快捷键（默认 `F9`）与「框选聊天区」快捷键。

### 翻译提示浮层（Toast）

- 可自定义背景色、文字颜色、背景透明度；
- 可设置显示时长（秒）与相对框选区域的位置（上方 / 右侧 / 下方，默认下方）。

### 设置界面与主题

- 设置窗口支持三种主题：**暗**（默认）、**浅**、**极**（赛博朋克风），一键循环切换；
- 暗色 / 赛博主题下标题栏跟随变暗；
- 控件圆角与布局优化；透明度改为自绘滑条，切换主题时颜色立即刷新；
- 本次打开设置窗口的位置会记住，关闭前不丢（不写入配置文件）。

### 快捷键

- 新增「设置窗口」快捷键（默认 `Home`），可快速打开 / 关闭设置；
- 允许绑定 `Home` / `End`；
- 托盘菜单精简，框选与 OCR 以快捷键与设置页操作为主。

### 问题修复

- 修复双语发送时，中文转译末尾字符显示不正确的问题。

## 项目来源

本项目是在 [BetterSCIME](https://github.com/typemiao/betterscime) 基础上修改和扩展的衍生项目。

BetterSCIME 提供了中文输入窗口、固定 `F8` 快捷键唤醒、IME 词表转译，以及通过模拟逐字键盘输入向游戏发送内容等基础功能。

本项目在 BetterSCIME 的基础上主要进行了以下修改和扩展：

- 增加快捷键设置功能；
- 将默认唤醒键调整为 `Enter`，使游戏聊天输入更加便捷；
- 支持设置其他按键作为输入窗口唤醒键；
- 将模拟逐字键盘输入改为通过剪贴板粘贴，提高发送效率和准确性；
- 增加游戏内截图到剪贴板功能；
- 增加截图快捷键设置功能；
- 增加系统托盘图标和右键菜单；
- 增加设置入口和软件退出功能；
- 调整并优化输入窗口及相关交互流程；
- 增加可选的中英双语发送（百度翻译 / 腾讯云 / 阿里云）；
- 设置中可切换翻译引擎、填写密钥，并提供开通与密钥获取链接及免费额度说明；
- 双语内容以一条消息、换行拼接的方式发送（`[zh] …` 换行 `[en] …`）；
- 增加聊天区框选、OCR 识别与英译中提示；
- 增加设置界面多主题与翻译提示外观配置；
- 每次启动软件时自动打开设置界面。

感谢 [BetterSCIME](https://github.com/typemiao/betterscime) 原作者 [typemiao](https://github.com/typemiao) 提供的基础代码和实现思路。

## 主要功能

- 使用系统输入法在叠加窗口中输入中文；
- 根据 IME 词表将中文内容转译为游戏可用文本；
- 通过剪贴板将转译后的内容粘贴至游戏聊天框；
- 可选中英双语发送：中文转译与英文翻译在同一条消息中换行发送；
- 支持百度翻译、腾讯云机器翻译、阿里云机器翻译三种引擎；
- 支持框选聊天区、OCR 识别英文聊天并翻译为中文；
- 支持自定义翻译提示浮层的颜色、透明度、时长与位置；
- 设置界面支持暗 / 浅 / 极（赛博）三种主题；
- 支持自定义输入窗口唤醒键、截图键、框选键、识别键、设置窗口键；
- 支持将游戏画面截图到剪贴板；
- 提供系统托盘图标、设置入口和退出功能；
- 启动时自动打开设置界面，便于首次配置。

## 使用方法

1. 运行 `SC-Tool.exe`，程序图标会出现在系统托盘中，并自动打开设置窗口。
2. 进入《星际公民》，按下输入窗口唤醒键，默认为 `Enter`。
3. 程序会打开游戏聊天框，同时显示中文输入窗口。
4. 切换至中文输入法并输入内容。
5. 再次按下 `Enter`：
   - 输入框中有内容时，程序会转译文本，通过剪贴板将内容粘贴至游戏聊天框并发送，然后关闭游戏聊天框和输入窗口；
   - 若已开启「双语发送」且密钥配置正确，同一条消息中会包含中文转译与英文翻译（中间换行）；
   - 输入框中没有内容时，程序不会发送消息，而是直接关闭游戏聊天框和输入窗口。
6. 按下 `Esc` 或点击输入窗口以外的区域，可以关闭输入窗口。
7. 在游戏中按下截图键，默认为 `F10`，可以将当前游戏画面截图到剪贴板。
8. 在设置中框选聊天区域后，按识别键（默认 `F9`）可 OCR 并英译中，结果出现在提示浮层中。
9. 按设置窗口键（默认 `Home`），或右键托盘图标选择“设置”，可修改快捷键、主题、翻译与提示外观等选项。
10. 右键单击托盘图标并选择“退出”，可以结束程序。

> 当设置的唤醒键不是 `Enter` 时，程序会自动模拟一次回车，以打开游戏聊天框。

## 双语发送（可选）

在设置中勾选「双语发送」，在 **翻译引擎** 下拉框中选择其一并填写对应密钥，可用「测试翻译」验证：

| 引擎 | 密钥字段 | 大致免费额度（以官方为准） |
| --- | --- | --- |
| 百度翻译 | APP ID / 密钥 | 身份认证后最高约每月 200 万字符 |
| 腾讯云翻译 | SecretId / SecretKey | 文本翻译每月约 500 万字符（需领取资源包） |
| 阿里云翻译 | AccessKeyId / AccessKeySecret | 通用版文本翻译每月约 100 万字符 |
| DeepSeek Flash / Pro | API Key | 按 token 计费（Flash 更便宜；Pro 更强），国内直连 |

设置界面中提供了各引擎的开通与密钥获取链接。发送时格式示例：

```text
[zh] …
[en] …
```

翻译失败时仍会发送中文转译；正常模式下会以短提示告知失败原因。

## 默认快捷键

| 功能 | 默认按键 |
| --- | --- |
| 唤醒输入窗口 | `Enter` |
| 截图 | `F10` |
| 识别聊天区 | `F9` |
| 打开 / 关闭设置 | `Home` |
| 框选聊天区 | 未设置（可在设置中绑定） |

## 配置文件

程序配置保存在 `SC-Tool.exe` 所在目录下的 `SC-Tool.cfg` 文件中，包括快捷键、截图提示选项、翻译提示外观、界面主题、是否开启双语发送，以及所选翻译引擎与密钥、聊天框选区域等。

## 构建

构建本项目需要安装 Rust 工具链。

在项目目录中执行：

```sh
cargo build --release
```

构建产物位于：

```text
target/release/SC-Tool.exe
```
