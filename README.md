# SC 小工具 v1.1.0

Star Citizen 小工具：中文输入叠加窗口 + 游戏内截图。第一次热键同时打开游戏聊天与本输入框，系统输入法打中文，第二次 Enter 发送或关窗。

## 用法

1. 运行 `SC-Tool.exe`（右下角托盘会出现图标）
2. 在 SC 里按唤醒键（默认 **Enter**）→ 游戏聊天框打开，同时弹出本输入窗
3. 切中文输入法，打字
4. 再按 **Enter**：
   - 有内容 → 转译并敲入游戏聊天 → 发送后关闭聊天与本窗口
   - 无内容 → 不发送，直接关闭游戏聊天与本窗口
5. **Esc** / 点别处 → 关输入窗
6. 游戏内按截图键（默认 **F10**）→ 截图到剪贴板
7. 托盘右键 **设置** → 可修改唤醒键 / 截图键（非 Enter 的唤醒键会自动模拟回车开聊天）
8. 托盘右键 **退出** → 结束程序

## 构建

```sh
cargo build --release
# 产物: target/release/SC-Tool.exe
```

配置保存在 exe 同目录的 `SC-Tool.cfg`。
